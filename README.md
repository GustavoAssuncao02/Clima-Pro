# ClimaPro React

Projeto React criado com Vite, Tailwind CSS e lucide-react.

## Como rodar

1. Abra a pasta do projeto no VS Code.
2. Abra o terminal dentro da pasta.
3. Rode:

```bash
npm install
npm run dev
```

4. Abra o link que aparecer no terminal, normalmente:

```bash
http://localhost:5173
```

## Comandos úteis

```bash
npm run build
npm run preview
```
